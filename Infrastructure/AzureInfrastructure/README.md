# Projet Infrastructure Azure

Ce projet simule le déploiement d'une infrastructure IaaS sur Azure.

### Description
- Création d'une machine virtuelle Linux simulée (VM-Pret)
- Région : EastUS
- Taille : Standard_B1s
- Utilisateur admin : azureuser

Ce projet fait partie de la chaîne CI/CD Azure DevOps pour héberger l'application Web ASP.NET Core `WebApp`.
